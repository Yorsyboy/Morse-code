# Project Name

> Decode a Morse code message


## Built With

- Ruby

## Getting Started

To get a local copy up and running follow these simple example steps.

Clone the project by using terminal:

```
git clone https://github.com/Yorsyboy/Morse-code.git
cd Morse-code
ruby decode.rb
```
